# Тест автоматической регистрации

Это тестовый файл для проверки работы автоматической регистрации в Telegraph Publisher.

## Что проверяем:
- Автоматическое создание аккаунта при отсутствии токена
- Сохранение токена в конфигурации
- Правильная обработка ошибок

## Результат:
Если система работает правильно, то при запуске команды публикации этого файла:
1. Будет создан новый аккаунт Telegraph
2. Токен будет сохранен в `.telegraph-publisher-config.json`
3. Файл будет успешно опубликован
